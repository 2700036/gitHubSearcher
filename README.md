# Поиск по GitHub репозиториям
  Главная особенность этого небольшого проекта в том, что пагинация реализована на чистом JS, без использования сторонних библиотек.
#  Установка:
# production
npm run build
# develop
npm run dev
# gh-pages
npm run deploy
